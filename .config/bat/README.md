# bat

`cat` replacement with syntax highlighting, used as the default pager throughout the shell config.

## Settings

| Option | Value |
|---|---|
| Theme | GitHub |
| Style | `changes, header, grid, numbers` |

Configured as the default `cat` via a Zsh alias (`cat` → `batcat`) and as the help pager (`H` global alias pipes `--help` output through bat).
