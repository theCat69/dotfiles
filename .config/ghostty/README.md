# Ghostty

GPU-accelerated terminal emulator configuration for Kubuntu.

## Settings

| Option | Value |
|---|---|
| Theme | Gruvbox Dark Hard |
| Font size | 12 |

## Keybindings

| Binding | Action |
|---|---|
| `Alt+Ctrl+Shift+↑` | Resize split up |
| `Alt+Ctrl+Shift+←` | Resize split left |
| `Alt+Ctrl+Shift+↓` | Resize split down |
| `Alt+Ctrl+Shift+→` | Resize split right |
