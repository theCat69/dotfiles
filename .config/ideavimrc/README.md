# IdeaVim

Vim emulation config for IntelliJ IDEA and other JetBrains IDEs via the [IdeaVim plugin](https://github.com/JetBrains/ideavim).

Config file: `.ideavimrc` (symlinked to `~/.ideavimrc`).

## Key Settings

- Leader key: `Space`
- Relative line numbers enabled
- System clipboard integration: `Ctrl+C` (copy in visual mode), `Ctrl+V` (paste)

## Keymaps

| Binding | Action |
|---|---|
| `<leader>rn` | Rename element |
| `<leader>gd` | Start debug |
| `<leader>qd` | Stop debug |
| `<leader>tf` | Select in project view |
| `<leader>aa` | Annotate (git blame) |
| `<leader>gh` | Show file history |
| `<leader>ne` | Go to next error |
| `<leader>bp` | Toggle breakpoint |
| `<leader>tt` | File structure popup |
| `<leader>/` | Find in file |
| `<leader>sg` | Find in path |
| `<leader>sf` | Go to file |
| `<leader>rf` | Refactoring menu |
| `<leader>?` | Recent files |
| `<leader>zz` | Hide all windows |
| `gc` (n/v) | Comment line |
| `Alt+Shift+K` | Move line up |
| `Alt+Shift+J` | Move line down |
