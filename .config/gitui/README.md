# gitui

Terminal UI for git — makes staging hunks, viewing diffs, and browsing history actually pleasant.

## Theme

Ships with all four [Catppuccin](https://github.com/catppuccin/catppuccin) variants:

| File | Variant |
|---|---|
| `frappe.ron` | Frappé (mid-dark) |
| `latte.ron` | Latte (light) |
| `macchiato.ron` | Macchiato (dark) |
| `mocha.ron` | Mocha (darkest) |

`theme.ron` is the active theme file — swap its contents with any of the above to change the active theme.

## Keybindings

Custom bindings in `key_bindings.ron` for muscle-memory consistency. `key_symbols.ron` defines the symbol set used in the UI.
